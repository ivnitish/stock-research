# Stock Research System

Personal fundamental research system for identifying multi-bagger opportunities in Indian equities (NSE/BSE) with selective US positions.

**Philosophy:** Charles Munger — read a lot, analyze deeply, hold long. Target: 25% IRR.

---

## What's Here

### Research Files (`research/`)
Per-company investment theses using a structured multi-bagger framework:
- Kill Filter → Compounding Engine → Management Quality → Competitive Landscape → Valuation
- Quality Score: 5 dimensions × 5 points = 25 max (A/B/C/D grade)
- DCF: Bear / Base / Bull with capacity-anchored growth rates

### Portfolio Dashboard
- **`output/html/portfolio.html`** — *single source of truth* for live portfolio state. Sortable table with qty, avg, CMP, P&L, % weight, target, upside %, action. Regenerated by `python3 src/refresh_portfolio.py --write`.
- **`output/html/index.html`** — research index across all coverage (held + watchlist). Has a portfolio snapshot strip at the top that links into portfolio.html.
- `PORTFOLIO_OVERVIEW.html` is now a redirect to portfolio.html (legacy URL); the old static markdown is archived under `research/archive/`.

Hosted at: [ivnitish.github.io/stock-research](https://ivnitish.github.io/stock-research)

### Source-of-truth chain
```
data/portfolio.csv (qty, avg)
  +  data/broker-exports/*.xlsx (CMP from broker close)
  +  output/html/index.html stock-rows (target, action label)
            ↓ src/refresh_portfolio.py
output/html/portfolio.html  +  index.html snapshot block
```
Notes:
- "Upside %" in portfolio.html is `(target / CMP − 1) × 100` — *no horizon assumed*. Targets in different research files use different horizons (1yr fair value vs 3-5yr base case), so a single "Expected 3yr CAGR" column was misleading and has been removed.
- "Avg Upside · Active Holds" in the snapshot is the current-weighted average upside across BUY/ADD/HOLD positions only (excludes EXIT/WATCH and unlabelled positions).

### Automation (`src/`)
- `refresh_portfolio.py` — single-command portfolio refresh from broker xlsx
- Daily portfolio price update (7pm)
- Weekly red flag monitor (promoter pledging, ROCE decline)
- Obsolete scripts archived under `src/archive/`: `portfolio_overview.py`, `portfolio_update.py`, `portfolio_review.py`

### BSE Filing Fetcher (`scripts/fetch_bse_filings.py`)
On-demand downloader for quarterly results, concall transcripts, annual reports, investor presentations from BSE.
```bash
python3 scripts/fetch_bse_filings.py KERNEX           # last 365 days
python3 scripts/fetch_bse_filings.py RAYMOND --days 180
python3 scripts/fetch_bse_filings.py ALL              # all portfolio stocks
```
Downloads to: `data/financial statemnt and concals/{SYMBOL}/`
